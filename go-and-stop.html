How to Go-Stop! Learn the Cards! Set Up!

"Go" & "Stop!"

When to use "GO" & "STOP":
  • The person who reaches the set amount of points can either “STOP” and end the game, or “GO” and continue the game further.
If you want to “GO” and continue the game:
  • If the winner makes the choice to “GO” once, then the winner gets one extra chip from each of the players.
  • If the winner makes the choice to “GO” twice, then the winner gets one extra chip from each of the player.
  • If the winner makes the choice to “GO” three times, the winner then gets double the chips from each player.
  • Now, if someone gets past three “GO”'s, each “go” after three times, the winner will get double chips from each of the players.
  • If you stop the game with no “GO”'s, you will receive the amount of chips of points that the game is set to, no extra chips from other players.
