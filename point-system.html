Point System!

Before playing the game, the players will need to decide how many points one needs to stop, in order to win the game. For a game with 2 players, the points are usually set to 5 to 7 points. For a game that has 3 players, the points are usually set to 3 points. However, this is all up for you to decide, based on how you want to play the game! The final score of the winner determines how much is collected from each player, if playing with chips or money. For example, if a player stops the game at 3 points, then each player gives the winner 3 of their chips or an equivalent amount of money that has been decided beforehand.

Any 3 Bright cards collected, equal 3 points

Any 4 Bright cards collected, equal 4 points

All 5 Bright cards collected, equal 15 points

Any 5 Animal cards collected, equal 1 point

Each Animal card that is collected beyond 5 cards is 1 point

However, if these 3 Animal cards are collected (Godori/고도리), it equals to 5 points

Any 5 Ribbon cards collected equal 1 point

Each Ribbon card that is collected beyond 5 cards is 1 point

If all 3 Blue Ribbons (Chungdan/청단) cards are collected, it equals to 3 points

If all 3 Red Ribbons (Hongdan/홍단) cards are collected, it equals to 3 points

If all 3 Red Plain Ribbons (Chodan/초단) cards are collected, it equals to 3 points

Any 10 Junk cards collected equal to 1 point

Each Junk card that is collected beyond the 10, are also 1 point
